# Postal Enhanced
## An improved mailbox addon for 3.3.5 wotlk environments.
Viva la Turtle🐢

Backported from Cata version and then upgraded for RP purposes.

- Send letters past the 500 character limit.
- Save letters without taking up a bank slot.
- Automatically saved sent letters.
- BlackBooks: Contact organizations
- Open All feature with multiple opening options
- Lots more.
